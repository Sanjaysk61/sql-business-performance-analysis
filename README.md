# Business Performance Analysis using SQL

This project analyzes revenue, customers, and product performance using advanced SQL techniques.

## Business Questions Answered
- How healthy is the business overall?
- Which products and categories drive revenue?
- Who are the most valuable customers?
- How does sales grow over time?
- How can customers be segmented for strategy?

## Skills Demonstrated
- CTE-based modular SQL design
- Window functions for running totals
- Business KPIs (AOV, growth, contribution)
- Customer segmentation
- Performance optimization

## Tech Stack
- SQL Server
- Dimensional Data Model
- Fact & Dimension Tables
